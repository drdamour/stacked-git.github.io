stg-version(1)
==============

NAME
----
stg-version - Print version information and exit

SYNOPSIS
--------
[verse]
'stg version' [OPTIONS]

DESCRIPTION
-----------

Print version information and exit

OPTIONS
-------
-s::
--short::
    Show abbreviated version information
