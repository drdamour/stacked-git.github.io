stg-goto(1)
===========

NAME
----
stg-goto - Go to patch by pushing or popping as necessary

SYNOPSIS
--------
[verse]
'stg goto' [OPTIONS] <patch>

DESCRIPTION
-----------

Go to patch by pushing or popping as necessary

OPTIONS
-------
-k::
--keep::
    Keep the local changes

-m::
--merged::
    Check for patches merged upstream
