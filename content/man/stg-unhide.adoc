stg-unhide(1)
=============

NAME
----
stg-unhide - Unhide hidden patches

SYNOPSIS
--------
[verse]
'stg unhide' [OPTIONS] <patch>...

DESCRIPTION
-----------

Unhide hidden patches in the series.

Hidden patches are no longer shown in the plain 'series' output.

OPTIONS
-------
-b <branch>::
--branch=<branch>::
    Use BRANCH instead of current branch
