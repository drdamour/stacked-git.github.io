stg-hide(1)
===========

NAME
----
stg-hide - Hide patches in the series

SYNOPSIS
--------
[verse]
'stg hide' [OPTIONS] <patch>...

DESCRIPTION
-----------

Hide patches in the series.

Hidden patches are no longer shown in the plain 'series' output.

OPTIONS
-------
-b <branch>::
--branch=<branch>::
    Use BRANCH instead of current branch
